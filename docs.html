Documentation | Artificial Neural Network Simulator

Create New Project

Run app.py > Click on "Create New Project" > Add project Details > Next > Add Network Details > Finish > Visualize the output

Open a project

Run app.py > Click on "Open a project" > Find the .cfg file > Open

Open Documentation

Run app.py > Click on "Open Documentation"

Main Window

  • To create a new project go to "File > New Project"
  • To open a project go to "File > Open Project"
  • To take snapshot of the output go to "File > Snapshot"
  • To exit go to "File > Exit"
  • To change dataset go to "Edit > Change Dataset"
  • To change parameters go to "Edit > Change Parameters"
